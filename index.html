TAGS

-DOCTYPE

Sirve para especificar el tipo de archivo que es (html) y se indican con < !DOCTYPE (el tipo de archivo)>

-TITLES

Se ponen mediante la expresión <title>(nombre de la página) </title>

-CABECERAS/HEADINGS

Se ponen mediante la expresión <h(1-6)> y se cierran con </h (el numero anterior) >. Para dar un tamaño especifico al heading se puede utilizar el estilo "font-size: ()px;"

-PARRAFOS

Se ponen mediante la expresión <p> y se cierran de la misma manera.

-LINKS

Se insertan mediante la expresión <a href="(enlace)">(link)</a>

-IMAGEN

Se insertan mediante la expresión <img src="enlace de la imagen" width=(anchura) height=(altura)>

-ALT

-ESTILOS

Se utilizan para dar un estilos como color, tamaño,... a un elemento. Se ponen con "style=(el estilo en concreto):(y la caracteristica)". Existen varios tipos de estilos, estos son unos de ellos:

  • Background color (style="background-color: (el color);")
  • Text color (style="text-color: (el color);")
  • Fonts (style="font-family: (el nombre del tipo de letra);")
  • Text size (style="text-size: (tamaño de la letra en pixeles (px));")
  • Text alignment (style="text-align: (center, left, right o justified);")

-TEXT FORMATTING

Estos son elementos para definir texto:

  • <b> - Negrita normal: buenos dias
  • <strong> - Negrita importante: buenos dias
  • <i> - Cursiva normal: buenos dias
  • <em> - Cursiva con importancia: buenos dias
  • <mark> - Para marcar palabras dentro de un texto sin marcar todo: buenos dias
  • <small> - Para hacer más pequeño algunas palabras dentro de un texto: buenos dias
  • <del> - Para tachar palabras: buenos dias
  • <ins> - Para definir elementos insertado o añadidos: buenos dias
  • <sub> - Para subscribir texto: buenos dias
  • <sup> - Para definir texto sobrescrito: buenos dias

-QUOTATIONS

Sirven para hacer citas o añadir informacion

  • <q> citas cortas:

    WWF's goal is to: Build a future where people live in harmony with nature.

  • <blockquote> citas extensas de otra pagina:
    For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization, WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally.
  • <abbr> abreviaturas:

    Este año el LoL cumple 10 años.

  • <address> para poner informacion sobre el autor de la página:
    Written by Unai Iraragorri.
    Visit us at: www.ejemplo.com
    Carretera La Estación Nº5, Fuenmayor
    Spain
  • <cite> para citar el nombre de un trabajo/obra:

    The Scream by Edvard Munch. Painted in 1893.

  • <bdo> para invertir la dirección en la que es escrita un texto:
    Este texto esta escrito de derecha a izquierda

-LANG

Sirve para declarar el idioma de la página y se pone cuando se abre el html: <html lang="sp-ES">

-TITLES (p)

Sirven para cuando se pase el rato.ón por encima del texto aparezca un mensaje. Se usa en el parrafo poniendo title="(lo que quieras poner)".

-HORIZONTAL RULES y LINE BREAKS (hr & br)

Sirven para poner una linea horizontal donde se ha puesto. Para crear la linea hay que poner <hr>. Los line breaks sirven para dejar una linea en blanco y se ponen con <br>.

-HEADS

Sirve como contenedor de metadatos. Se pone <head> después de html y antes del title y </head> antes del body.

-PRE

Sirve para escribir un texto preformateado. Se utiliza igual que la p pero con pre. 

 (Este texto utiliza pre)
  My Bonnie lies over the ocean.

  My Bonnie lies over the sea.

  My Bonnie lies over the ocean.

  Oh, bring back my Bonnie to me.

-COLORES

  • Background color (style="background-color: (el color);")
  • Text color (style="text-color: (el color);")
  • Border color (style="border:(tamaño en px) (el color);")
  • Color values:
    • RGB: se combinan rojo, verde y azul para hacer colores dando un valor a cada uno de 0 a 255:

      rgb(60, 179, 113)

    • HEX: igual que el rgb pero con valor hexadecimal, se utiliza poniendo #rrggbb (rr=red, gg=green, bb=blue) con valores de 00 a ff:

      hex #3cb371

    • RGBA: funciona igual que el rgb pero con el elemento alpha que indica la transparencia;

      rgba(60, 179, 113, 0.5)
